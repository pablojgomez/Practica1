Position


La position propiedad especifica el tipo de método de posicionamiento utilizado para un elemento (estático, relativo, absoluto, fijo o adhesivo).
position = static|absolute|fixed|relative|sticky|initial|inherit;



Width:auto

La width propiedad establece el ancho de un elemento.
¡El ancho de un elemento no incluye relleno, bordes o márgenes!
Nota: Las propiedades min-width y max-width anulan la width propiedad.
Valor por defecto: auto
width: auto|value|initial|inherit;



Height: 100px: Establezca la altura de un elemento al 100% de la altura del elemento principal:

Definición y uso: La height propiedad establece la altura de un elemento. La altura de un elemento no incluye relleno, bordes o márgenes.
Si height: auto;el elemento ajustará automáticamente su altura para permitir que su contenido se muestre correctamente.
Si heightse establece en un valor numérico (como píxeles, (r) em, porcentajes), entonces si el contenido no cabe dentro de la altura especificada, se desbordará. La propiedad de desbordamiento define cómo manejará el contenedor el contenido desbordado .
Nota: Las propiedades min-height y max-height anulan la heightpropiedad.
Valor por defecto: auto
